# as-bayraklari

An initiative, using frames from [`as-bayraklari`](https://github.com/erdemoflaz/as-bayraklari), to allow any computer with `curl` to display an _animated_ turkish flag.

# Try it!
```bash
node index.js
curl localhost:3000
```
